# JEAF X-Fun Implementation README #

This repository contains the default implementation of JEAF X-Fun API. This default implementation should be suitable for almost all cases. 

## Links ##
For further information please refer to:

* [JEAF X-Fun Getting started](https://anaptecs.atlassian.net/l/c/8vuqYGsN)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.x-fun/jeaf-x-fun-impl)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.x-fun/jeaf-x-fun-impl)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
